# HWP Studio 기능 안내

## 시작하기 전에

1. **한글(HWP) 프로그램을 먼저 실행**하세요 (빈 문서 열기)
2. `/hwp-setup`으로 환경을 확인하세요

## 사용 가능한 커맨드

| 커맨드 | 설명 | 예시 |
|--------|------|------|
| `/hwp-setup` | 환경 진단 + 설치 가이드 | "Python, 한글 설치 확인" |
| `/hwp-fill` | 양식 자동 채우기 | "과업지시서 빈칸 채워줘" |
| `/hwp-write` | 문서 처음부터 작성 | "사업계획서 작성해줘" |
| `/hwp-analyze` | 문서 분석 + 요약 | "이 문서 내용 요약해줘" |
| `/hwp-convert` | 형식 변환 (PDF/DOCX) | "PDF로 변환해줘" |
| `/hwp-batch` | 다건 문서 일괄 생성 | "엑셀 명단으로 위촉장 생성" |
| `/hwp-privacy` | 개인정보 스캔 | "개인정보 있는지 확인해줘" |
| `/hwp-help` | 이 안내 페이지 | "뭘 할 수 있어?" |

## 자주 하는 작업

### "양식에 내용을 채우고 싶어요"
→ `/hwp-fill` 사용. 파일 경로와 참고자료를 알려주세요.

### "처음부터 문서를 작성하고 싶어요"
→ `/hwp-write` 사용. 문서 종류와 양식을 알려주세요.

### "문서 내용을 보고 싶어요"
→ `/hwp-analyze` 사용. 파일 경로만 알려주세요.

### "특정 텍스트를 바꾸고 싶어요"
→ "OO를 XX로 바꿔줘"라고 자연어로 말하면 됩니다.

### "다른 형식으로 저장하고 싶어요"
→ `/hwp-convert` 사용. PDF, DOCX, HTML 지원.

## 한글 파일을 열어야 하나요?

**네, 대부분 한글 프로그램이 실행 중이어야 합니다.** 그냥 한글을 켜두세요. 문제가 생기면 플러그인이 자동으로 안내합니다.

## HWP vs HWPX

- **HWPX 권장**: 텍스트 검색/치환이 더 안정적입니다
- 변환: 한글에서 "다른 이름으로 저장" > HWPX 선택
