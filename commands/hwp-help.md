# HWP Studio — 한글 문서 AI 자동화 (94개 도구)

## 시작하기

1. **한글(HWP) 프로그램을 먼저 실행**하세요 (빈 문서 열기)
2. `/hwp-setup`으로 환경을 확인하세요
3. 아래 커맨드 중 필요한 것을 선택하거나, 자연어로 바로 요청하세요

## 커맨드 목록

### /hwp-fill — 양식 자동 채우기
HWP/HWPX 양식의 빈칸을 AI가 자동으로 채웁니다.
- **사용 예시**: "과업지시서.hwp 채워줘", "사업계획서 빈칸 채워줘"
- **참고자료**: Excel/CSV/PDF/DOCX/PPT 모든 형식 지원
- **주요 기능**: 서식 보존, 라벨 자동 매칭, 참조 데이터 .md 저장
- **팁**: 참고 엑셀 파일이 있으면 함께 알려주세요 — 자동 매핑됩니다

### /hwp-write — 문서 처음부터 작성
공문, 사업계획서, 보고서, 동의서 등을 처음부터 작성합니다.
- **사용 예시**: "사업계획서 작성해줘", "인사발령 공문 만들어줘"
- **짧은 문서** (1~4쪽): 한번에 완성
- **긴 문서** (5쪽+): 목차 먼저 → 섹션별 분할 작성
- **팁**: 양식 파일이 있으면 서식을 그대로 따릅니다

### /hwp-analyze — 문서 분석 + 요약
문서의 구조, 표, 텍스트를 분석하고 요약합니다.
- **사용 예시**: "이 문서 내용 요약해줘", "표 구조 파악해줘"
- **주요 기능**: AI 심층 분석, 표 데이터 추출, 서식 프로파일 추출

### /hwp-convert — 형식 변환
HWP를 PDF, DOCX, HTML 등으로 변환합니다.
- **사용 예시**: "PDF로 변환해줘", "DOCX로 내보내줘"
- **지원 형식**: HWP ↔ HWPX, HWP → PDF, HWP → DOCX, HWP → HTML

### /hwp-batch — 다건 일괄 생성
Excel/CSV 데이터로 위촉장, 증명서 등을 대량 생성합니다.
- **사용 예시**: "명단.xlsx로 위촉장 100건 생성해줘"
- **주요 기능**: 자동 매핑, 파일명 규칙 지정, PDF 일괄 변환
- **팁**: 양식 파일 + 데이터 파일 경로를 알려주세요

### /hwp-privacy — 개인정보 스캔
문서에서 주민번호, 전화번호, 이메일 등을 자동 감지합니다.
- **사용 예시**: "개인정보 있는지 확인해줘"

### /hwp-setup — 환경 진단
Python, pyhwpx, 한글 프로그램 설치 상태를 자동 확인합니다.
- **사용 예시**: "환경 확인해줘", 에러 발생 시 자동 안내

## 커맨드 없이 자연어로도 됩니다

```
"C:/문서/사업계획서.hwp 열어서 분석해줘"
"표의 계약금액 칸에 50,000,000원 채워줘"
"OO를 XX로 바꿔줘"
"PDF로 변환해줘"
"참고자료.xlsx 읽어서 양식 채워줘"
```

## 지원하는 참고자료 형식

| 형식 | 방식 |
|------|------|
| Excel (.xlsx/.csv) | 직접 읽기 — 표 데이터 자동 매핑 |
| PDF | PyMuPDF 텍스트 추출 |
| Word (.docx) | PDF 변환 또는 python-docx 직접 추출 |
| PPT (.pptx) | PDF 변환 또는 python-pptx 직접 추출 |
| TXT/JSON/MD | 직접 읽기 |
| 이미지 (PNG/JPG) | Claude 멀티모달 시각 분석 |

- **환경/문서 관리** (6): 열기, 닫기, 저장, 환경 체크
- **문서 분석** (16): 구조 분석, 표 조회, 텍스트 추출, 마크다운 변환
- **텍스트 편집** (18): 삽입, 검색/치환, 서식, 각주/미주, 페이지 번호
- **표 편집** (18): 생성, 셀 채우기, 병합, 배경색, 테두리, 수식
- **페이지/레이아웃** (9): 여백, 머리글/바닥글, 다단, 시각 검증
- **서식/그리기** (5): 스타일 적용, 글상자, 선, 캡션
- **스마트/복합** (16): AI 분석, 자동 채우기, 간트차트, 일괄 생성
- **HWPX 전용** (4): 템플릿, 빈 문서 생성, XML 편집
- **내보내기** (2): DOCX, HTML 내보내기

## HWP vs HWPX

- **HWPX 권장**: 텍스트 검색/치환이 더 안정적입니다
- 변환: 한글에서 "다른 이름으로 저장" > HWPX 선택
- 한글 프로그램이 실행 중이어야 합니다
